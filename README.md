## Tech Stack

**Frontend**
- React 19, React Router v7
- Vite, TailwindCSS
- Framer Motion, Three.js
- Axios, jsPDF

**Backend**
- Node.js, Express 5
- MongoDB + Mongoose
- JWT Authentication
- bcrypt, Helmet, Rate Limiting

## Getting Started

### Backend
```bash
cd mess-manager-server
cp .env.example .env   # fill in your values
npm install
npm run dev
```

### Frontend
```bash
cd mess-manager-client
cp .env.example .env   # fill in your values
npm install
npm run dev
```
